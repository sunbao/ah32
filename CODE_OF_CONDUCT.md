# 行为准则（阿蛤 / AH32）

本项目采用《Contributor Covenant 行为准则》（贡献者公约）。

## 我们的承诺

为了营造开放、友善、无骚扰的环境，我们承诺：无论年龄、体型、可见或不可见残障、种族、性别特征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、相貌、种姓、肤色、宗教或性取向，我们都会为每一位参与者提供积极体验。

## 我们的标准

鼓励的行为包括但不限于：

- 尊重不同观点与经验
- 接受建设性批评
- 以项目整体最佳利益为出发点
- 对社区成员保持同理心

不可接受的行为包括但不限于：

- 侮辱/骚扰性言论与人身攻击
- 公共或私下骚扰
- 未经许可发布他人的私人信息
- 其他不恰当或不专业的行为

## 责任与执行

维护者有权并有责任澄清并执行行为准则，并对任何他们认为不当、威胁、冒犯或有害的行为采取适当且公正的纠正措施。

如需报告问题，请通过 `SECURITY.md` 中的渠道联系维护者。
