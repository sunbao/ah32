# Vue 3 + TypeScript + Vite

该目录是前端（WPS TaskPane）工程，基于 Vue 3 + TypeScript + Vite。
模板使用 Vue 3 的 `<script setup>` 单文件组件（SFC）语法；更多细节可参考 [script setup 文档](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup)。

关于推荐的项目结构与 IDE 支持，参见 [Vue TypeScript 指南](https://vuejs.org/guide/typescript/overview.html#project-setup)。
